# Ruboty::NpbReport

## Installation

Add this line to your application's Gemfile:

```ruby
gem "ruboty-npb_report"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-npb_report

## Usage

    ruboty npb report
